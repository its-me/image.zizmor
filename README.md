# zizmor

[zizmor](https://github.com/woodruffw/zizmor) packaged as a multi-arch Docker image.

## Tags

| Tag | Description |
|---|---|
| `latest` | Latest zizmor release |
| `X.Y.Z` | Specific zizmor version |

## Platforms

`linux/amd64` · `linux/arm64` · `linux/ppc64le`

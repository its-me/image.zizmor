# zizmor

[zizmor](https://github.com/woodruffw/zizmor) packaged as a multi-arch Docker image.

Source: [github.com/its-me/image.zizmor](https://github.com/its-me/image.zizmor)

## Tags

| Tag | Description |
|---|---|
| `latest` | Latest zizmor release |
| `X.Y.Z` | Specific zizmor version |

## Platforms

`linux/amd64` · `linux/arm64` · `linux/ppc64le`

- [GitHub Container Registry](https://ghcr.io/its-me/zizmor)
- [Docker Hub](https://hub.docker.com/r/1tsme/zizmor)

## GitHub Action

[its-me/action.zizmor](https://github.com/its-me/action.zizmor) is a GitHub Action that runs zizmor using this image.
